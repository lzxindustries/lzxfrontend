# Video Blending Matrix

## Overview

The video blending matrix is a versatile signal processing module used for mixing input signals in various mathematical input and output modes. cascaded input columns allow each of the vbm’s three 5-channel dc mixers to operate either independently or combined in various input matrix configurations. signal inputs:each mixer has five dc-coupled signal inputs which can be used for any type of signal, from low frequency modulation to full bandwidth video. four have associated level/inversion controls and inversion mode switches, the remaining one is a direct summing input for chained operation be
