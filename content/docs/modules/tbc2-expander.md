# TBC2 Expander

TBC2 Expander is a **companion front-panel** for [TBC2](/modules/tbc2). It provides additional video input connections—today primarily **VGA** for both decoder channels, so you can use **YPbPr over VGA** as used on many video mixers and other gear. Cables to link the expander to the main TBC2 module are included with the product.

## Relationship to TBC2

The core decoding, frame sync, genlock, encoders, and UI live on the main **TBC2** module. The expander only brings signals to the edge of the case through the connectors on this panel. Full behavior, input standards, and firmware are documented in the [TBC2 manual](/modules/tbc2/manual): read that reference for what each input can do, how the decoders and genlock work, and how to update firmware.

## Physical installation

Install the TBC2 Expander in the same row as your TBC2, connect the included **ribbon or harness** to the TBC2 module per the current assembly instructions shipped with the hardware. Keep cable runs tidy and power down the case before re-routing the link between modules if you are moving modules in the frame.
