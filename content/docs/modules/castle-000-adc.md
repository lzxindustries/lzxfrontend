# Castle 000 (ADC)

Castle is the LZX **CMOS digital** series for 5V logic and open-hardware DIY builds, designed alongside **Cadet** (analog) and **Gen3** (full modular). **000** is a dual **8-bit analog-to-digital** converter, turning analog voltages into **digital** patterns you can process with the rest of the **Castle** matrix.

## Role in a patch

Use the Castle ADCs when you need to **code** the strength of a CV or video-referenced voltage into a digital stream for gates, counter inputs, and combination logic **downstream** in a Castle or Cadet + Castle hybrid patch.

## Power and build

This module is designed for the **+5V** and **-5V** supplies used across the **Castle** series, with a **+12V** line as documented on the build guide for your **revision** of the hardware. Build from the **official BOM and artwork**; match parts to the current repository release before ordering a run of boards.
