# Double Vision Expander

The Double Vision line uses the same **TBC2 Expander**-style **VGA** input hardware described for the [TBC2 Expander module](/modules/tbc2-expander/manual). Use that module manual for **panel** layout, **cables** to **TBC2**, and **YPbPr over VGA** practice. The [TBC2](/modules/tbc2/manual) module manual remains the **system** reference for decoders, **firmware**, and **Media** / **ramp** behavior.

The [Double Vision](/instruments/double-vision) page summarizes the current retail system. Existing Double Vision owners looking at a **case** or **expander** upgrade should follow the **contact** path on the **store** or newsletter announcements you received with your product.
