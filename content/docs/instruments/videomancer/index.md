# Videomancer Manual

Reference documentation for the Videomancer performance video
instrument. Every section here is complete technical reference — use
the sidebar to jump between topics.

For a curated path through the material (quick-start, learning
sequence, deep-dive picks) visit the
[Videomancer Learn page](/instruments/videomancer/learn). This page
is the manual's table of contents, not the product landing.
