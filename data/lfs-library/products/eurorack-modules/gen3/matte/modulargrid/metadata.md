## Description

**Static voltage generator / color picker**

Matte is a DC offset generator optimized for selection of RGB colors.

Each of the three channels has an A output and a B output, controlled by the potentiometers. Range is 0 to +1 volts.

Additionally, each channel features an "A+B" output. This is the average value of the A and B outputs, not the sum.
