## Description

**Analog to Digital Converter**

The Analog to Digital Converter is used to change an analog input signal in to a 3-Bit data stream. Bias and Gain (+/-N) are used to scale and bias the input signal. Use this module to interface with the rest of the Castle series or as a hard zone colorizer by itself.
